Overview

Plugin for generating skid site investigation reports.

Each site has up to 1 report and corresponds to 1 feature on a layer.

How to use

1:Set the layer

Import a suitable layer. The layer needs:

  • A text,int or float field named site
  • A text field named csv

    • Set layer to this.

    2:Surveyed/Unsurveyed

    The Surveyed/Unsurveyed button will change the layer style to display surveyed and unsurveyed features differently.(rule based, features with a value in the "csv" field are considered to be surveyed).

    3:Changing site

    Changing site will load an existing report or reset the form. This will lose any unsaved changes.

    The plugin will look in the csv in the "csv" field of the corresponding feature.

    Site can be changed through:

    • Dropdown
    • Previous site button (<)
    • Next site button (>)
    • Selecting 1 feature on the layer,right clicking dropdown and "change to selected feature"

    4:Fill the form

    Values not in dropdowns can be temporarily added by typing in value and pressing enter.

    5:Save the form

    Click save... button to save. Reports are output as csv files with 1 row per site. Multiple sites can be saved to the same csv. Saving will set the "csv" field of the feature.
